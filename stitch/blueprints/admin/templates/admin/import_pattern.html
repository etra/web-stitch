## Field reference

**Required fields:**
- "name": pattern title
- "description": short description
- "width" / "height": grid dimensions (10–100 stitches)
- "colors": maps single-character aliases to DMC thread codes. Use null for empty cells (typically ".").
- "grid": array of strings, one per row (height rows, each width characters). Each character is a color alias. This grid places **full cross stitches**.

**Optional stitch arrays (for detail work):**
- "three_quarter_stitches": array of objects with x, y, color (alias), corner ("tl", "tr", "bl", "br"). A three-quarter stitch covers 2 quadrants. Two **opposite** corners at the same cell (tl+br or tr+bl) can coexist with different colors.
- "half_stitches": array of objects with x, y, color, direction ("/" or "\\"). A half stitch is one diagonal line across the cell.
- "quarter_stitches": array of objects with x, y, color, corner ("tl", "tr", "bl", "br"). A quarter stitch goes from the named corner to the cell center.
- "back_stitches": array of objects with startX, startY, endX, endY, color. These are lines (outlines/details) connecting grid **corners** (intersections). Coordinates are 0..width for X and 0..height for Y (inclusive — corners, not cells).

## Rules
- The grid handles full cross stitches. Leave cells as "." where you want partial stitches instead, then specify them in the appropriate array.
- All color aliases used in any field must be defined in "colors".
- Use real DMC color codes. Common ones: 310 (Black), blanc (White), 321 (Red), 699 (Green), 796 (Royal Blue), 444 (Dark Lemon), 740 (Tangerine), 550 (Violet), 3799 (Charcoal), 415 (Pearl Gray), 898 (Coffee Brown), 307 (Lemon), 666 (Bright Red), 602 (Cranberry), 907 (Light Parrot Green), 814 (Dark Garnet), 3041 (Medium Antique Violet), 3761 (Light Sky Blue), 739 (Ultra Very Light Tan).
- Max 128 unique colors.
- three_quarter_stitches, half_stitches, quarter_stitches, and back_stitches are all optional — only include them when the design needs detail or outlines.
